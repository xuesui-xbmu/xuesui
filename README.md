# Web前端学习 Skill · 最终导出版

## 一、Skill 基础信息

- **Skill ID**：web-frontend-learning

- **Skill 名称**：Web前端学习技能树

- **Skill 类型**：学习规划 / 能力自检 / 知识体系 / 面试辅助

- **适用对象**：零基础入门者、在校学生、前端初学者、面试备考者、技能提升者

---

## 二、功能定义

基于企业真实前端开发标准构建的**完整、结构化、可自检、可执行**的技能体系 Skill。

将前端学习路径拆分为**基础核心 → 工程化框架 → 进阶提升 → 实战能力 → 工具软技能**五大模块，覆盖从入门到就业的全流程知识点，支持学习规划、能力评估、查漏补缺、简历优化、面试考点梳理、项目技术评估。

---

## 三、技能结构明细

### 1. 基础核心层

- HTML：语义化标签、表单与表格、多媒体、SEO 结构、可访问性规范

- CSS：选择器与优先级、盒模型、Flex/Grid 布局、响应式布局、动画过渡、BEM 命名、常见布局问题

- JavaScript：基础语法、函数与作用域、闭包与 this、数组对象方法、Promise/async/await、DOM 操作、事件机制、本地存储、正则基础

### 2. 工程化 & 框架层

- 工程化：Git 版本管理、npm/pnpm 包管理、Vite/Webpack、ESLint/Prettier、跨域解决方案

- 框架：Vue3 / React 基础与 Hooks、路由管理、状态管理、UI 组件库使用

- 小程序：原生小程序开发、uni-app / Taro 跨端开发

### 3. 进阶提升层

- TypeScript：基础类型、接口、泛型、类型约束

- 网络与优化：HTTP/HTTPS、浏览器缓存、性能优化、代码分割、图片优化

- 效率工具：Sass/Less、Axios 封装、JWT/Token 登录鉴权

- 拓展能力：ECharts 可视化、Three.js 基础、Node.js 基础

### 4. 实战能力层

- PC 端与移动端页面开发

- 后台管理系统搭建

- 商城、博客、小程序等完整项目开发

- 接口联调、权限控制、路由守卫

- 项目部署：Nginx、GitHub Pages、Vercel

### 5. 工具与软技能

- Chrome DevTools 调试

- Figma / PS 设计稿还原

- 需求分析与组件拆分

- 文档阅读与问题排查能力

---

## 四、使用场景

1. 零基础入门，明确学习顺序，避免盲目自学

2. 学习进度自查，定位薄弱环节并针对性补强

3. 配合课程学习，标记掌握程度，形成学习地图

4. 面试前系统梳理考点，覆盖高频核心知识点

5. 快速生成专业简历技能描述

6. 项目开发前进行技术栈能力评估

## 五、触发指令

- `/skill 前端路线`：生成 1/3/6 个月学习计划

- `/skill 前端自检`：能力评估，生成掌握/薄弱清单

- `/skill 面试考点`：输出高频前端面试题

- `/export`：导出 Skill 完整结构化文档
